# Autonomous-Mental-Health-Agent-
An AI agent that provides empathetic mental health support, detects crisis situations, searches for resources, and generates personalized wellness plans — built with Python, Groq (LLaMA 3.3), and Streamlit.
I used Python as the core language. For the AI itself, I used Groq's API to run the LLaMA 3.3 model — I chose Groq specifically because of its speed and free tier. For the interface, I used Streamlit, since it let me build a full web app in pure Python without touching HTML or JavaScript. To make it an actual agent rather than just a chatbot, I integrated DuckDuckGo's search API as a callable tool, so it can decide to search the web when a user needs resources. And I used python-dotenv to keep my API key secure instead of hardcoding it
